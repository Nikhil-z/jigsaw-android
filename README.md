# Jigsaw (J 拼图) for Android

![banner](http://7xqspp.com1.z0.glb.clouddn.com/16-5-17/25004728.jpg)

这是一个简易的拼图游戏，玩家可以选择内置的或自己相册中的图片开始游戏，游戏将记录玩家完成所花时间以及移动的步数。
